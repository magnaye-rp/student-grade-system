# Student Grade System

A simple web-based student grade management system.

## Features

- User authentication (login/logout)
- Dashboard with statistics
- Grade management for students
- Responsive design

## Installation

1. Place the files in your web server directory
2. Access the system through your browser
3. Default login credentials:
   - Username: admin
   - Password: admin123

## File Structure

- `index.php` - Landing page
- `login.php` - Login form
- `dashboard.php` - Main dashboard
- `grades.php` - Grade management
- `css/style.css` - Stylesheet
- `js/app.js` - JavaScript functionality

## Usage

1. Open `index.php` in your browser
2. Click "Login" to access the system
3. Use the dashboard to navigate to different sections
